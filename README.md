# SymBnb 

## AirBnb clone with Symfony 4


### System Requirements

PHP => 7.4.9

Symfony 5 

Npm or Yarn 

Composer

Local server

## Project setup
```

yarn install

composer install 

php bin/console doctrine:database:create

php bin/console make:migration

php bin/console doctrine:migrations:migrate 

yarn encore dev --app

```

### Run App

symfony server:start
```
